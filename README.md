#  LinkedIn auto fill Form

Using LinkedIn JavaScript API for User Authentication and Profile Retrieval.


## Prerequires

1. http-server


## Run

Clone

```
git clone https://github.com/humbertodias/js-linkedin-auto-fill-form.git
```

Inside

```
cd js-linkedin-auto-fill-form
```

Start Server

```
http-server -p 9999
```

Browser

```
http://localhost:9999
```
